01

비타민C 세럼
소비자 조사

17만 개 판매

정성 편향 인식 → 정량 추가 설계 · ANOVA 검증 · 제품 전략 전면 반영

편향 인식 → 정량 추가 설계 → ANOVA 검증 → 제품 반영 → 역대 최고 데뷔

자세히 보기 ▾

배경

기존 비타민C 라인의 매출 부진 및 고객 경험 저하로 리뉴얼 결정. 소비자 조사 설계부터 분석까지 전 과정을 단독으로 수행.

리서치 설계

정성조사(화상 심층 인터뷰) 진행 후, 기존 결론의 편향을 인식. 저관여 소비자를 포함한 대규모 정량조사(n=2,000↑)를 추가 설계.

핵심 인사이트

  • 시술 관련 키 클레임이 컨셉 중 가장 유효함을 ANOVA로 검증
  • 소비자가 고농도로 인지하는 최소 농도는 10% — 마케팅 메시지 기준으로 확립
  • 초기 정성 결론(고농도 선호)은 고관여 집단의 편향이었음

의사결정 연결

두 인사이트 모두 최종 비타민 C 시리즈에 전면 반영되었으며, 마케팅 키 클레임 결정에 기여함. 이러한 성과를 인정받아 정규직으로 전환되었음.

결과: 일본 Qoo10 메가와리에서 17만 개 이상 판매, 역대 최고 데뷔 실적 달성.

02

젤크림 카테고리 전환 &
좌담회 단독 기획

출시 직전

사전조사 기반 카테고리 전환 제안 → 3주 리드타임 좌담회 전 과정 단독 수행

사전조사 분석 → 카테고리 전환 제안 → 좌담회 단독 운영 → 출시 전략 수립

자세히 보기 ▾

배경

신제품 세럼의 소비자 인터뷰 과정에서, 개발된 제형이 기존 제품군(세럼)이 아닌 다른 카테고리로서 소비자가 원하는 속성을 충족한다는 사실을 발견.

의사결정 연결

여름용 젤크림으로의 카테고리 전환을 제안, PM·마케팅 전원 동의 하에 채택. 카테고리 변경에 따른 추가 사용성 검증이 필요해 2차 좌담회를 직접 기획.

실행 과정

  • 3주 리드타임, 에이전시 없이 전 과정 단독 수행
  • 자사 온드미디어로 일본인 고객 직접 리크루팅·선별·컨택
  • 장소 예약, 다과, 포스터 세팅 등 현장 전 과정 직접 운영
  • 일본어 모더레이팅으로 고객과 직접 소통, 만족 요인 도출

결과: 현재 출시를 앞두고 있으며, 인플루언서의 사전 테스트에서 긍정적인 평가를 받음.

03

일본 소비자 분별
척도 개발

전사 도입

사내 전례 없는 척도를 R로 직접 개발 → 전사 기본 스크리너 도입 · 팀 발령

구조적 문제 발견 → R 알고리즘 개발 → 파일럿 검증 → 전사 도입

자세히 보기 ▾

문제 정의

설문 10건 이상 수행했으나 결과 간 비교 불가. 표준화 체계 부재로 데이터는 쌓이는데 인사이트가 누적되지 않는 구조적 문제 발견.

접근 방식

사내에 통계 지식 보유 인력 부재. 학부 심리통계 지식을 상업용 리서치에 적용하는 도전을 직접 시작.

실행 과정

  • 기존 소프트웨어로 구현 불가 → R로 점수 산출 알고리즘 직접 하드코딩
  • 파일럿 설문 응답 데이터 기반으로 가산점 반복 수정
  • 정규분포 확인 후 완성도 검증

결과: 이후 모든 설문의 기본 스크리너로 전사 도입. 독립 설문 간 결과 비교 가능해져 전략 의사결정 고도화. 마켓 인텔리전스팀 데이터 큐레이터로 발령.

핵심 역량

고객 인터뷰 & 좌담회

심층 인터뷰(IDI) 설계·진행, 좌담회 기획·모더레이팅 (일본어 직접 진행)

정량 조사 설계·분석

대규모 설문 설계 (n=2,000↑), ANOVA · 회귀분석, R · SPSS · Excel 활용

시장·경쟁사 분석

일본 뷰티 시장 트렌드 분석, Best Practice 도출, 현지 리서치사 협업

비즈니스 인사이트 도출

인사이트 → 제품 전략 반영, SKU 결정 기준 수립, 카테고리 전환 제안 및 채택
