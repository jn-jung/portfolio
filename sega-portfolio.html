PROJECT 01

ビタミンCセラム 新商品開発プロジェクト

ANUA · 日本事業本部

背景と課題

既存ビタミンC製品ラインの売上低迷と品質問題によるCX疲労度の増加。リニューアルに向けたマーケットリサーチおよび消費者調査をインターン課題として担当し、正社員転換後は製品発売まで全プロセスを主導しました。

担当範囲（DRI）

競合デスクリサーチ → Best Practice導出 → デプスインタビュー（定性）→ 大規模アンケート設計・実施（定量、n=2,000以上）→ R/SPSS統計分析 → 発売/未発売SKUの意思決定基準策定

正社員転換後は、現地リサーチ会社ASMARQを通じた消費者インタビューおよびHUT（Home Use Test）運営で最終サンプルを確定。

コアインサイト

定性調査で導出された「高濃度志向」結論の偏りを認識し、低関与消費者を含む大規模定量調査を追加実施。ANOVA分析によりコンセプト別選好度を検証し、施術関連のキークレームが最も有効であることを立証。また、消費者が「高濃度」と認識する最低濃度が10%であることを導き出し、マーケティングメッセージの基準として確立しました。

両結論ともに最終製品に全面反映 — データが製品を作り上げたプロジェクト

PROJECT 02

ヒアルロン酸カテキン ジェルクリーム — カテゴリー転換＆座談会

ANUA · マーケットインテリジェンスチーム

危機的状況

日本GTMチームからの要請で開発したヒーローセラムの消費者調査結果、「セラムとしては購入しない」という結論に。製品発売中止の危機 → 下半期の新商品空白というクリティカルな事態が発生。

大胆な提案

事前調査データを再分析し、カテゴリーをセラムから夏用ジェルクリームへ転換することを提案。PMチームとGTMチーム双方の合意を獲得しました。

3週間で成し遂げたこと

リサーチ会社を活用する時間的余裕がない状況で、全プロセスを単独で遂行：

✦ 自社メディアを通じた日本人顧客の直接リクルーティング・選定
✦ 座談会会場の手配、ポスター・告知文の制作、茶菓・ギフトのセッティング
✦ PMチーム向けの通訳手配、現場メイクアッププログラムの企画
✦ 日本語モデレーティングによる使用性テスト実施・インサイト導出

座談会参加者がSNSで接客への好評レビューを投稿 → ブランドイメージ向上にも貢献

成果

座談会で改善した製品属性が日本KOL先行提供で高評価を獲得。さらに東南アジア市場でも好評を得て、日本専用ではなくグローバルSKUへと拡大。

PROJECT 03

消費者性向 標準化尺度の開発

データ基盤の意思決定体制構築

問題認識

入社後10件以上の大規模アンケートを実施するも、結果を統合的に解釈することが不可能に。標準化された体系がないため、データは蓄積されてもインサイトが蓄積されない構造的課題を発見しました。

チャレンジ

社内に統計知識を持つ人材が皆無な環境で、大学時代のCASI翻案・妥当化プロジェクトでの知見を基に心理統計理論を実務リサーチに応用。既存ソフトウェアでは実装不可能だったスコアリング体系をRで直接ハードコーディングして開発しました。

インパクト

以後、全社アンケートの基本スクリーナーとして採用。独立していた調査結果を同一基準で比較可能にし、新商品コンセプト検証やプロモーションターゲット設定など、戦略的意思決定の精度が大幅に向上しました。

本成果により、リサーチ統括のマーケットインテリジェンスチーム・データキュレーターに配属
